# Portafolio-de-juegos

Este proyecto es una muestra de mis capacidades como **Frontend Developer Junior**.

### 🚀 Características Técnicas:
* **Consumo de APIs:** Integración en tiempo real con **RAWG API** para datos de videojuegos.
* **Avatares Dinámicos:** Uso de **DiceBear API** para generación de perfiles de usuario.
* **Diseño Responsivo:** UI/UX inspirado en interfaces de consolas modernas usando **Bootstrap 5** y **CSS3** personalizado.
* **Lógica en JS:** Manejo de asincronía (`Async/Await`), filtrado dinámico de arreglos y manipulación del DOM.

### 🛠️ Tecnologías:
HTML5 | CSS3 | JavaScript (ES6+) | Bootstrap 5 | FontAwesome
